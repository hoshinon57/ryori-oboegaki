# ryori-oboegaki
2026年4月から週に1回程度のペースで、妻に細かく教わりながら料理をしている。  
といっても現状は「料理している」とは言えないレベルだが。  
水の量やいためる順など、細かいメモを残していく。

----

チャーハンメモ
冷凍チャーハンではなく、具や米を自分で調理するやつ
★途中から妻にバトンタッチしたので、前半部分のみ
・油　適量　大さじ3～4杯ぐらいかな
・ひき肉　300g
・卵　2個
・たまねぎ　1/2個　みじんぎり
・野菜ミックス　1/2袋

たまねぎ→卵→肉→野菜ミックス、の順にいためる。
※やったのはここまで。
　この後、妻がお米を入れていためて完了、、、だったかと思う。調味料あったかも。
　妻いわく「肉や野菜などの具材は自由。何入れても美味しい」とのこと。
