# ryori-oboegaki
2026年4月から週に1回程度のペースで、妻に細かく教わりながら料理をしている。  
といっても現状は「料理している」とは言えないレベルだが。  
水の量やいためる順など、細かいメモを残していく。

----
肉じゃがメモ
食材：
・じゃがいも 3個（大きいのなら2個）
・人参 1/2個ちょいで十分 皮むきを忘れないように
・玉ねぎ　1個
・牛肉切り落とし（薄切り肉）　250gぐらいの、サニーだと「大サイズ」で売ってるやつかな
・糸こんにゃく　2パック
方針：
ベターホーム本に沿って作る。
（糸こんにゃくは本には書かれていない点に注意）
水や調味料は、本の2倍量で作る。水なら200→400g、料理酒は大さじ1⇒大さじ2、など。煮る時間は本は15分だが、同じぐらいで良さそう、じゃがいもの具合を見ながら。
糸こんにゃくはざるにあけて熱湯をかける。これでアク抜き、味が染みやすくなる、などになるらしい。
糸こんにゃくは、鍋にて肉のアク抜きをした後に入れる。アク抜き前に入れると、糸こんにゃくにアクがからまってしまうとのこと。

薄味だったので、調味料（しょうゆ）をもう少し多めでも良かったかも。
(5/E)
　上記の「2倍量」「しょうゆ多め」を意識しても、作った当日のは味が薄めだった。ただし追加しょうゆ等をせずとも、翌日はかなり味がしみててさらに美味しかった。
　次は「アクを取った後に水を足さない」など検討する。

----
カレーメモ

・じゃがいも 3～4個
・人参 1/2個ちょいで十分 皮むきを忘れないように
・玉ねぎ　1個
・肉：
　牛豚ひき肉のときは、400gほどだった
　※鶏肉だったら、同じように400gか、"1羽ぶん"取りたいなら400g弱でもOK.
・カレールー
　大人用：「6人前」として使う。
　　　　　1箱12人前で箱に2パック入っている。
　　　　　なので、2種の箱からそれぞれ「1パックの半分」ずつ取ると、6人前相当になる。
　子ども用：「3人前」として使う。
　　　　　1箱に2パック入っているので、「1パックの半分」で。

油は多めと言われた。おおさじ3～4杯か、もうちょいかな。
肉→野菜の順にいためる
大人と子供用の鍋に分けるのは、火がとおって水も適量いれてコトコト煮込んだ後。目分量でよいので、子供用のなべに全体の2/3をうつす。
水が蒸発してどろっとしすぎると、子どもらにとっては辛いイメージがあるようなので、良い感じに水を足すのが良い。

----
そうめんメモ
・そうめん　揖保乃糸のやつ(1パック300g)
　※くらしモアは安いが、まずい。買わないこと。
　※今回は別メーカーので350g使った。十分に食べきれる。
　　揖保乃糸だと300g入りなので、もう1パック買って使うか、お惣菜を多めに。
・めんつゆ（ヤマキ）
・きゅうり、ハム、ミニトマト
　※ハムは1人2枚
・お惣菜（イカフライがおすすめ）
・氷を作っておく

基本はそうめんのパックに書かれている通りに作る。
水切り、水洗いをした後にボウルに入れ、氷を乗せると良い。
野菜やお惣菜は別皿に入れて、各自で取ってもらう。

★途中から妻にバトンタッチしたので、前半部分のみ

・ひき肉　300g
・卵　2個
・たまねぎ　1/2個　みじんぎり
・野菜ミックス　1/2袋

たまねぎ→卵→肉→野菜ミックス、の順にいためる。
